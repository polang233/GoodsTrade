# GoodsTrade - 简单安全的玩家物品交易插件

> 一个专为 Minecraft 服务器设计的轻量级物品交易插件，提供安全、便捷的玩家间交易功能。

![示例](/img/QQ20260315-190201-HD.gif "交易界面演示")

## 📖 简介

GoodsTrade 是一个简单易用的玩家交易插件，支持可视化 GUI 界面操作。玩家可以安全地交换物品，避免交易诈骗。

### ✨ 特性

- 🛡️ **安全保护**：支持交易期间免疫伤害和行动限制
- 🎯 **可视化界面**：直观的 GUI 操作，无需复杂命令
- ⏱️ **确认机制**：双方确认后倒计时，确保交易安全
- 🔒 **物品锁定**：确认后无法更改交易物品
- ⚙️ **可配置**：支持自定义等待时间、触发方式等

## 🚀 命令

| 命令                      | 权限               | 说明          |
|-------------------------|------------------|-------------|
| `/gt trade [玩家名]`       | `gt.trade.use`   | 向指定玩家发起交易   |
| `/gt trade [发起者] [接收者]` | `gt.trade.admin` | 强制两个玩家进行交易  |
| `/gt accept`            | `gt.trade.use`   | 接受当前交易请求    |
| `/gt accept [玩家名]`      | `gt.trade.use`   | 接受指定玩家的交易请求 |
| `/gt reload`            | `gt.trade.admin` | 重载插件配置文件    |

---

## 🔐 权限节点

| 权限               | 默认   | 说明                 |
|------------------|------|--------------------|
| `gt.trade.use`   | true | 允许使用交易功能           |
| `gt.trade.admin` | op   | 管理员权限，可使用强制交易和重载命令 |

## 🎮 使用方法

### 基础交易流程

1. **发起交易**：
    - 方式一：输入命令 `/gt trade [玩家名]`
    - 方式二：潜行状态下右键点击玩家（需配置开启）

2. **接受交易**：
    - 点击聊天栏中的 `[点击接受]` 链接
    - 或输入命令 `/gt accept`

3. **放置物品**：将想要交易的物品放入交易界面左侧（发起者）或右侧（接收者）

4. **确认交易**：点击绿色按钮确认，双方都确认后进入倒计时

5. **完成交易**：倒计时结束后，物品自动交换

### 💡 交易提示

- ✅ 确认后将锁定物品，无法再修改
- ⏰ 双方都确认后开始 5 秒倒计时
- ❌ 倒计时期间可取消，回到初始状态
- 🎒 交易取消或关闭界面时，物品自动返还
